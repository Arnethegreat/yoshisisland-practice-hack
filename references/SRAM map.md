Yoshi's Island has 32KB of Save RAM, spanning from $700000~$707FFF. This map explains what each address represents in standard gameplay [gamemode](https://github.com/Raidenthequick/yoshisisland-disassembly/wiki/Game-Modes) $0F. The reason so much important stuff is stored here is because the Super FX can only access cartridge RAM, not SNES system RAM. Note that other gamemodes may use these addresses for completely different purposes, but as of now it is only $0F being documented (possible future plans to include more gamemode maps).  
This map migrated from [SMWCentral](https://www.smwcentral.net/?p=nmap&m=yisram). You can also check the [main memory domain, WRAM](https://github.com/Raidenthequick/yoshisisland-disassembly/wiki/WRAM-Map).
