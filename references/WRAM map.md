The Super Nintendo Entertainment System has 128kb of system RAM (aka WRAM) that every game can access, spanning from $7E0000~$7FFFFF. This map explains what each address represents in Yoshi's Island standard gameplay gamemode $0F. In Yoshi's Island in particular, since it has the Super FX enhancement chip, much of the game's memory is stored in [cartridge RAM](https://github.com/Raidenthequick/yoshisisland-disassembly/wiki/SRAM-Map) instead. This is why there is so much free WRAM in this game. Nonetheless, there are still some pretty important pieces of data stored here.
